settings Setup Guide: Routing Desktop/Music Audio to Browser Sources

01

Add a Browser Source

In Streamlabs Desktop or OBS Studio, click the + under Sources and select Browser Source.

02

Critical Audio Options

Paste your URL. Make sure to check the box that says "Control Audio via OBS" (or Streamlabs). This redirects your audio output to trigger the visual waves!

03

Set Resolution & Scale

Set the dimensions to match your stream output (e.g., 1920x1080) to ensure crisp renders with pixel-perfect lines.
